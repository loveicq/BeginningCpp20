# Markdown 语法速查表

## 基础语法
1. **标题**  
   `# H1`  
   `## H2`  
   `### H3`  
   `#### H4`

2. **文本样式**  
   - 斜体：`*斜体*` → *斜体*  
   - 粗体：`**粗体**` → **粗体**  
   - 删除线：`~~删除线~~` → ~~删除线~~

3. **列表**  
   - 无序列表：  
     `- 项目1`  
     `- 项目2`  
   - 有序列表：  
     `1. 第一项`  
     `2. 第二项`

4. **链接与图片**  
   - 链接：`[文本](https://example.com)`  
   - 图片：`![描述](图片路径.jpg)`

5. **代码**  
   - 行内代码：`` `code` `` → `code`  
   - 代码块：  
     ```  
     \```语言  
     代码块  
     \```  
     ```  
     （示例：\```python）
